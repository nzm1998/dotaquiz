# CLAUDE.md - Dota2 题目网站升级任务

## 项目背景

这是一个 Dota2 趣味答题网站，位于 `~/Desktop/nzmdota2project`。
当前版本：基础答题功能，20道题，单页HTML应用。

## 本次升级任务（2026-05-11）

### 目标
将网站升级为更高级的 UI + 新功能版本。

### 具体需求

#### 1. UI 高级化
- 参考高端游戏网站设计风格
- 深色主题 + 金属质感/Ovilee 电竞风 + 霓虹点缀
- 主色：Dota2 红 `#e94560`，金色 `#f0a500`，深色背景 `#0a0a12`
- 字体：Oxanium（标题）+ Noto Sans SC（正文）
- 卡片式布局，圆角 + 毛玻璃效果
- 悬停动画、过渡动效
- 选项要有明显的 hover/selected 状态变化

#### 2. 难度选择页
- 答题前先显示难度选择页
- 两个按钮：「初学者」和「老刀斯林」
- 初学者：全部20题
- 老刀斯林：全部20题（但高难度题目比例更高，可随机打乱题序）

#### 3. 选项随机顺序
- 每题加载时随机打乱 A/B/C/D 顺序
- 同一题刷新页面顺序可能不同

#### 4. 答题后显示本题正确率
- 答完后显示：「本题正确率：XX%」
- 正确率数据用 localStorage 存储（每题：答对次数/总回答次数）
- 格式：「本题正确率：73%（共100人作答）」

#### 5. 称号系统（最终结果页）
根据正确率给予称号：
- >= 90%：「👑 老刀斯林」—— 真正的刀塔传奇
- 80-89%：「🔥 真刀斯林」—— 经验丰富的老玩家
- 60-79%：「🛡️ 刀斯林」—— 不错的刀斯林
- 40-59%：「🤔 假刀斯林」—— 云玩家实锤了
- 20-39%：「☁️ 云玩家」—— 你真的打过刀塔吗？
- < 20%：「💀 云玩家本云」—— 你怕不是只看过视频吧

称号要在结果页大字显示，并配一段点评文字。

#### 6. 音效
- 答对：清脆的 correct 音效（可用 Web Audio API 生成或找开源音效 URL）
- 答错：低沉的 wrong 音效
- 用户可静音

#### 7. BGM
- 使用 Valve 官方 Dota2 主界面音乐
- 可在网上找 Dota2 main menu music 的 CDN 链接或使用开源资源
- 页面顶部或右下角有 BGM 开关按钮（默认开启）
- 图标表示：🔊 开启 / 🔇 静音

## 架构说明

- 依然是单 HTML 文件（index.html）内嵌所有代码
- CSS/JS 直接写在 HTML 内
- questions.json 保持不变（20道题都保留）
- 正确率数据用 localStorage 存储，key 格式 `q1_correct`（整数）等
- 音频：使用 HTML5 Audio，src 用公开 URL
- BGM：使用 HTML5 Audio autoplay loop，src 用公开 URL

---

## 页面流程

1. **难度选择页** → 选「初学者」或「老刀斯林」
2. **答题页**（展示题目 + 选项 + 进度条）
3. **单题结果页**（答完立即显示对错 + 正确率）
4. **最终结果页**（称号 + 点评 + 正确率统计 + 分享按钮）

## 开发进度（2026-05-16）

### ✅ 已完成

1. **UI 高级化**
   - 深色主题 + 渐变卡片 + 悬停动效
   - 响应式布局，支持移动端

2. **难度选择页**
   - 初学者和老刀斯林两个入口
   - 题目随机抽取，打乱顺序

3. **答题流程**
   - 选项随机打乱（A/B/C/D 顺序不固定）
   - 答题后立即显示对错
   - 下一题按钮在答题后出现

4. **正确率统计（localStorage）**
   - 每题回答后写入 localStorage
   - 格式：`q{id}_correct` 和 `q{id}_total`
   - 答题后显示本题正确率

5. **称号系统**
   - 6个等级：老刀斯林 → 云玩家本云
   - 结果页显示称号 + 点评文字

6. **音效**
   - Web Audio API 生成答对/答错音效
   - 可静音

7. **评论区（localStorage）**
   - 每道题独立评论存储
   - key 格式：`comments_q{id}`
   - 最多保存20条，按时间倒序

### localStorage 数据结构

```javascript
// 正确率统计
q1_correct = "3"    // 答对3次
q1_total = "5"      // 总共回答5次

// 评论区（每题独立）
comments_q27 = [
  { text: "这道题真难！", author: "快乐的刀斯林", timestamp: 1778861306533 }
]
```
